JQuery UI

Datepicker

Aquí se muestra el widget de jQuery UI datepicker, que permite mostrar una interfaz de calendario cuando se pincha sobre el input para introducir la fecha.

Lo he implementado con el formato de fecha español corto (dd-mm-YYYY), es decir dos dígitos para el día, dos para el mes y cuatro para el año.
